# Certificate Generator Server

We will use `Flask` to build our server and `opencv-python` for certificate processing

_Note: All needed packages are encapsulated inside `requirements.txt` file_

## Setup Guide

Install `virtualenv`
```sh
pip install virtualenv
```

For Windows:
```sh
git clone https://github.com/shecodesvietnam/certificate-generator-back.git
cd certificate-generator-back
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

For Linux:
```sh
git clone https://github.com/shecodesvietnam/certificate-generator-back.git
cd certificate-generator-back
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
